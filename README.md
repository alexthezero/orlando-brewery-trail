# Orlando Brewery Trail Checklist

A simple static checklist web app for tracking breweries around Orlando, Florida.

## Files

- `index.html`
- `style.css`
- `app.js`

## Features

- Brewery checklist cards
- Progress counter and progress bar
- Search
- Filters for Orlando, nearby bonus stops, food, visited, and not-yet visited
- Notes per brewery
- Google Maps links
- Website links
- Local progress saving with `localStorage`
- Exportable checklist text

## GitHub Pages setup

1. Create a new GitHub repository. Suggested name: `orlando-brewery-trail`
2. Upload `index.html`, `style.css`, and `app.js`
3. Go to Settings > Pages
4. Set source to `main` branch and `/root`
5. Save
6. Your site will publish at:
   `https://YOUR-GITHUB-USERNAME.github.io/orlando-brewery-trail/`

## Editing breweries

Open `app.js` and edit the `breweries` array at the top.
